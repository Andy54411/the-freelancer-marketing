## Project Name: Tasko AI Business Operating System & Marketplace

### Google Ads API Antrag - Begründung

Die Google Ads API ist entscheidend, um unseren **Marktplatz** zu befeuern und unseren Dienstleistern beim Wachstum zu helfen. Wir benötigen die API für:

1. **Automatisierte Lead-Generierung:** Ermöglicht Dienstleistern das Schalten lokaler Anzeigen (z. B. "Notdienst"), die Traffic direkt auf ihr **Tasko-Marktplatz-Profil** oder Buchungsformulare leiten.

2. **Closed-Loop-Attribution:** Durch die Verbindung von Google Ads mit unserem **Rechnungssystem** verfolgen wir, ob ein Anzeigenklick zu einer bezahlten Rechnung geführt hat, und liefern so präzise ROAS-Daten, die über einfaches Conversion-Tracking hinausgehen.

3. **Intelligentes Kapazitätsmanagement:** Automatisches Pausieren von Anzeigen, wenn die **Zeiterfassung** oder der **Kalender** eines Anbieters anzeigt, dass er ausgebucht ist, um verschwendete Ausgaben für nicht bedienbare Leads zu vermeiden.

4. **Nahtloses Onboarding:** Nutzung von OAuth zur Verknüpfung bestehender Werbekonten, damit Anbieter ihr Marketing verwalten können, ohne die Plattform zu verlassen, die sie für den täglichen Betrieb nutzen.

> Diese Version deckt nun wirklich **alles** ab: ERP, Marktplatz, B2B/B2C-Interaktion und die spezifischen Vorteile der Integration. Das sollte für Google sehr überzeugend sein.
